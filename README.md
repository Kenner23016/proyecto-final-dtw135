# Inventario Básico de Productos

Proyecto Final DTW135 - Avance 1.

## Tema asignado

Opción 4: Inventario básico de productos.

## Estructura usada

```txt
proyecto-final-dtw135/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
├── workers/
├── assets/
└── docs/
```

## Qué contiene este avance

Este avance está limitado a lo trabajado en las prácticas iniciales:

- HTML5
- CSS básico
- JavaScript
- Objeto principal `app`
- Arreglos de objetos
- Funciones
- Eventos con `addEventListener`
- Delegación de eventos
- Validaciones básicas
- Manipulación del DOM
- `localStorage`
- `sessionStorage`
- Cookies

## Funciones implementadas

- Agregar productos
- Listar productos en una tabla
- Editar productos
- Eliminar productos
- Guardar productos en `localStorage`
- Guardar el encargado usando cookies
- Guardar estado de sesión usando `sessionStorage`
- Cambiar tema claro/oscuro usando `localStorage`
- Mostrar resumen básico del inventario:
  - Productos registrados
  - Unidades en stock
  - Valor total del inventario

## Pendiente para próximos avances

Todavía no se implementa lo siguiente:

- `try/catch`

- API REST
- Geolocalización
- Dashboard avanzado
- Reporte final

1. Abrir la carpeta `proyecto-final-dtw135` en VS Code.
2. Abrir `index.html`.
3. Ejecutar con Live Server o abrir en el navegador.

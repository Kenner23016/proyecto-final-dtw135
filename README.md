# Inventario Básico de Productos

Proyecto Final DTW135 - Avance 1

## Tema asignado

Opción 4: Inventario básico de productos.

## Descripción

Este avance contiene una aplicación web frontend para registrar, consultar,
editar y eliminar productos de un inventario básico.

La estructura se organizó en varias vistas para que el proyecto se vea más
ordenado y parecido a una aplicación tradicional:

- `index.html`: inicio y resumen del inventario.
- `pages/productos.html`: listado y gestión de productos.
- `pages/producto-form.html`: formulario para agregar o editar productos.
- `pages/sesion.html`: encargado, sesión y tema.
